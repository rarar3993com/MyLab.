Pelajari Lebih Lanjut

    • HTML

      • HyperText Markup Language (HTML) adalah sebuah bahasa standar yang digunakan mempublikasikan dokumen secara online sehingga bisa diakses, menampilkan berbagai informasi didalam sebuah penjelajah web internet dan pemformatan hipertext sederhana yang ditulis dalam berkas format ASCH agar dapat meghasilkan tampilan wujud yang terintegrasi.

    • CSS

      • Cascading Style Sheet merupakan fitur untuk membuat dynamic HTML, style sheet yang mendeskripsikan tampilan dokumen HTML dan membuat special effect. Terdapat tiga cara dalam penggunaan CSS, pertama adalah inline style sheet, kedua adalah internal style sheet dan yang terakhir adalah eksternal style sheet.
